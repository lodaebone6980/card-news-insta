# 브랜딩 전략서

## 1. 브랜드 코어

### 미션
> 복잡한 뉴스와 트렌드를 누구나 이해할 수 있는 비주얼 콘텐츠로 전환하여, 바쁜 현대인이 빠르게 핵심을 파악할 수 있도록 돕는다.

### 비전
> 카드뉴스 하나로 "아, 이거였구나"라는 순간을 매일 만든다.

### 핵심 가치
1. **명확성** — 어렵게 말하지 않는다. 한 장에 하나의 메시지.
2. **시의성** — 지금 가장 화제인 것을 다룬다.
3. **접근성** — 전문가가 아니어도 이해할 수 있다.
4. **일관성** — 브랜드 톤과 비주얼이 매번 동일하다.

## 2. 타겟 오디언스

### 1차 타겟
- **연령**: 25~39세
- **직업**: 직장인, 프리랜서, 1인 창업자
- **특성**: 트렌드에 관심 있지만 긴 글을 읽을 시간이 없음
- **소비 채널**: 인스타그램, 스레드

### 2차 타겟
- 해당 분야에 진입하려는 학생/취준생
- 동료에게 트렌드를 공유하고 싶은 사람

### 페르소나 (brand.md에서 사용자가 커스텀)
- 이름, 나이, 직업, 고민, 콘텐츠 소비 패턴을 brand.md에 정의

## 3. 시장 내 위치

### 포지셔닝
"매일 아침 커피 한 잔과 함께 보는 3분 카드뉴스"
- 속도: 빠른 소비 (5장 이내)
- 깊이: 표면적이지 않되, 논문 수준은 아닌 "적정 깊이"
- 톤: 전문가의 지식 + 친구의 말투

### 경쟁 대비 차별점
| 기존 카드뉴스 | 우리 |
|---|---|
| 디자이너가 수동 제작 | AI 자동 생성 → 속도 |
| 브랜드 톤 불일치 | brand.md 기반 일관성 |
| 주제 선정이 감에 의존 | RSS + AI 큐레이션 |

## 4. 브랜드 인격

### 성격
- 똑똑하지만 겸손한
- 호기심 많고 탐구적인
- 따뜻하고 배려하는

### 말투 원칙
- "~입니다" 대신 "~거든요", "~인 셈이에요"
- 전문 용어 사용 시 반드시 쉬운 설명 병행
- 이모지 사용 지양 (깔끔함 유지)
- 단정적 표현 대신 "~라고 해요", "~로 보여요"

## 5. 비주얼 터치포인트

### 컬러 시스템
- brand.md의 메인/서브 컬러를 따름
- 본문 텍스트: #1A1A1A (거의 블랙)
- 보조 텍스트: #6B7280 (그레이)

### 타이포그래피
- 제목: 볼드, 큰 사이즈
- 본문: 레귤러, 가독성 우선
- 강조: 메인 컬러 또는 볼드

### 레이아웃 원칙
- 1080 x 1350px (인스타 세로 최적)
- 여백 충분히 (최소 60px 패딩)
- 한 슬라이드 = 하나의 시각적 초점

## 6. 사용자와의 약속
> "매일 3분, 당신이 놓치고 있던 트렌드를 가장 쉽게 전달합니다."
