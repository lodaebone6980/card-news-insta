STEP 0

브랜드 설정 — 한 번만 해두면 끝

앱 좌측 하단 "브랜드 설정" → 아래 항목 채우기 → 저장

브랜드명: 내 계정명 (예: @devdesign.kr) 인스타핸들: devdesign.kr 메인 컬러: #FF6B35 ← 포인트 컬러 (오렌지 추천) 서브 컬러: #0a0a0a ← 배경 컬러 (다크 테마 필수) 타겟 독자: AI·자동화에 관심 있는 2030대 톤앤매너: 구어체, 단호하게, 어려운 말 X CTA 문구: 팔로우하면 계속 옵니다 →
왜 중요한가

브랜드 설정이 없으면 AI가 매번 다른 스타일로 생성해요. 한 번 설정해두면 모든 카드뉴스가 동일한 톤&컬러로 나옵니다.

STEP 1

어떻게 콘텐츠를 넣을까

4가지 방법 중 상황에 맞게 선택하세요.

🌐

오늘의 뉴스 가져오기

RSS에서 오늘 가장 핫한 뉴스를 AI가 자동으로 선정해서 카드뉴스로 만들어줌.

✓ 매일 꾸준히 올릴 때 — 클릭 한 번으로 끝
📋

텍스트 붙여넣기

블로그 글, 보고서, 아이디어 메모 등 어떤 텍스트든 붙여넣으면 카드뉴스로 변환.

✓ 내가 쓴 글을 카드뉴스로 만들 때
🔗

URL 가져오기

특정 기사나 웹페이지 URL을 입력하면 그 내용을 자동으로 추출해서 카드뉴스로.

✓ 특정 기사를 카드뉴스로 만들 때
📡

피드에서 선택

구독 중인 RSS 피드에서 여러 기사를 보고 직접 골라서 카드뉴스 제작.

✓ 주제를 직접 고르고 싶을 때
STEP 2

카드뉴스 만드는 전체 흐름

처음부터 끝까지 이 순서로 하면 됩니다.

1

브랜드 설정 (최초 1회)

앱 좌측 하단 ⚙️ → 브랜드 설정. 브랜드명, 메인 컬러(#FF6B35), 서브 컬러(#0a0a0a), 타겟 독자, 톤앤매너, CTA 입력 후 저장.

EXAMPLE

메인 컬러: #FF6B35 (오렌지) · 서브 컬러: #0a0a0a (다크) · CTA: 팔로우하면 계속 옵니다 →

2

콘텐츠 선택

4가지 방법 중 하나를 선택합니다. 가장 빠른 건 "오늘의 뉴스 가져오기" — 클릭 한 번으로 오늘 날짜 뉴스를 자동 수집.

3

"카드뉴스 만들기" 클릭

AI가 자동으로 주제 선정 → 기획 → 리서치 → 글쓰기 → 이미지 생성 → 슬라이드 렌더링까지. 약 1~3분 소요.

내부에서 일어나는 일: 1. RSS 크롤링 → 오늘 헤드라인 수집 2. Gemini가 주제 1개 선정 3. 5~7장 기획안 작성 4. 팩트/통계 리서치 5. 최종 카피 작성 (브랜드 톤 반영) 6. 슬라이드별 이미지 생성 7. 1080×1350 PNG 렌더링
4

오른쪽 프리뷰에서 확인 & 수정

폰 프리뷰에서 슬라이드를 좌우로 넘기며 확인. 마음에 안 드는 슬라이드는 텍스트를 직접 수정하고 재렌더링 가능.

5

다운로드 → 인스타 업로드

PNG 파일로 다운로드 후 인스타그램에 슬라이드 형식으로 업로드. 완료!

TIP

인스타그램 업로드 시 첫 장이 커버로 보이니 cover.html 디자인이 가장 중요합니다.

FAQ

자주 묻는 질문

막히는 부분이 있으면 여기서 먼저 확인하세요.

카드뉴스가 생성이 안 되고 에러가 나요
GEMINI_API_KEY가 설정되어 있는지 확인하세요. Railway 대시보드 → Variables에서 GEMINI_API_KEY 환경변수가 있어야 합니다.
배경이 흰색으로 나와요 (다크 테마가 안 돼요)
브랜드 설정에서 서브 컬러를 #0a0a0a로 바꿔주세요. 기본값 #F5F3FF는 흰색입니다.
슬라이드 레이아웃을 내 스타일로 바꾸고 싶어요
GitHub의 templates/ 폴더 안에 있는 HTML 파일 8개를 수정하면 됩니다.
뉴스 소스를 내 분야로 바꾸고 싶어요
GitHub의 news_sources.txt 파일에 RSS URL을 추가/수정하면 됩니다.
인스타그램에 자동 업로드도 가능한가요?
현재는 수동 업로드만 지원합니다. n8n이나 Make를 연결하면 자동화 가능합니다.
